  1. Create an html and a css file in the repository by typing gedit filename & in terminal
  2. Edit the html file and css file and validate the code
  3. While editing, view your work on your development server by typing python -m SimpleHTTPServer in terminal, and typing in localhost:8000 on the URL bar on the web
  4. When you finish editing, save the files and type in sftp (your computer's username)-(the computer you are transfering to). When you type this command in terminal, made sure you cd to your repository folder
  5. When you go to sftp, type in your password and then cd to your public html folder
  6. type this command put filename to upload the files from your computer to sftp
  7. Now that the file is saved in the public html folder, type in cmis-hsteacher/~username1234 in the URl bar on the web
